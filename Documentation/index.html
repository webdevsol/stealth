Installation

Salad Template is fully responsive, simple, clean and mobile friendly coming soon page with nice countdown timer, parallax background, subscribe form and contact form.
Its build based on Twitter Bootstrap, so it has many features.
It is easy to install and customize. In general, you can create unlimited number of themes with our Compass/SASS Script. Salad Template is compatible with all major browsers on desktop, tablets and smartphone devices!

1. Installation

Copy next files and directory to your root directory.
To run Salad Template your host just needs PHP 5.0 or later. 

/index.html
/assets directory
/subscribe.php
/mailer.php

2. Setup

To change logo image just replace current image with yours in same size.
Logo image is located at assets/gfx/logo.png.

File # Property Description
index.html line 20 countdownDate Counterdown limit date in format yyyy/mm/dd HH:ii:ss (2013/06/24 23:45:00)
index.html line 21 animeTime Time duration for all animations on page
index.html line 22 openAtInit Auto-Open one of your panels (Subscribe, About or Contact)
index.html line 23 initTimeout Wait before auto-open one of your panels
index.html line 24 initPanelToOpen Panel to open on startup - can be "subscribe", "about" or "contact"
core.scss line 4 trans Opacity value for all panels (0-1) eg 0.75
core.scss line 5 subscribecolor Color for subscribe panel
core.scss line 6 about Color for about panel
core.scss line 7 contact Color for contact panel
core.scss line 7 textshadow Color for text shadows

2.1. How to change Color Theme?

Requirements

  • Ruby
  • Compass
  • SASS

Install ruby.
How to install Compass and SASS

Open file /compass/sass/core.scss and edit color values to your favorite colors.
Open cmd/terminal, go to compass-style directory and compile new theme. 

compass compile

2.2 How to change background parallax image

Open index.html file and edit line 31.

2.3. How to set subscribers list storage file?

Open subscribe.php file and edit line 3.

$file_name = 'subs.txt';

If you want to receive email for every subscribsion, please uncomment line 42 and enter your information

@mail('your@email.com', 'New Item Added', 'Added email address: ' . $email);

2.4. How to set your / receiver email address?

Open mailer.php file and edit line 3.

$to_email = 'email@domain.com';
